# AOC2023

My solutions to Advent of Code 2023, written entirely in Julia

## Usage

Run code for solutions to problems for the selected day
```bash
$ julia --project=. run.jl <day>
```

Run tests
```bash
$ julia --project=. test/runtests.jl [day]
```
